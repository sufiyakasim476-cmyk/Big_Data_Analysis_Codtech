# Big_Data_Analysis_CodtechOverview
This task demonstrates big data processing using tools like PySpark and Dask. The goal was to analyze large datasets and show how distributed computing can handle scalability.

Deliverables
Jupyter Notebook / Script with big data operations

Dataset (Sales, Employees, AI‑Data)

Insights: sales trends, employee distribution, and category performance

Tools Used
PySpark – Distributed data processing

Dask – Parallel computing in Python

Jupyter Notebook – Interactive analysis

Highlights
Processed large datasets with filtering, grouping, and aggregation

Derived insights such as top categories and performance metrics

Showed how big data frameworks scale better than traditional pandas workflows
